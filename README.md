# poup-backend

Back-end desenvolvido em Spring Boot para o projeto POUP, focado em educação financeira acessível.

Este repositório representa a versão estável do back-end, utilizada em conjunto com o front-end ativo e funcional. Ele foi criado a partir da consolidação do trabalho da equipe, garantindo um ambiente íntegro, operante e alinhado ao escopo final do projeto.

A versão completa do projeto, juntamente com os commits dos integrantes, está disponível em:
https://github.com/Matheussaares/POUP-2

A não utilização do repositório citado como versão final se deve a complicações técnicas encontradas no back-end daquela implementação. Após diversas tentativas de correção, a equipe optou por seguir com este repositório, no qual obtivemos sucesso pleno tanto no back-end quanto no front-end.
